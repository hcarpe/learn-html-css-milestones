A List of Basic HTML Tags:

  • Paragraph Tags

    The paragraph tag is used to separate different paragraphs in an HTML document's text

  • Blockquote Tags

    The blockquote tag is used to add blockquotes to an HTML document.

    Blockquotes are usually indented and specifies a section that is quoted from another source.

    Text usually surrounds blockquotes, and you need to provide the source of the quote using the blockquote cite attribute within the tag. You can find the source by inspecting the blockquote element from the browser.

  • Pre-Formatted Text Tags

    Other tags don't usually doesn't preserve white space, but the pre tag preserves white space and line breaks in text.

  • i and Em Tags

    The em and i tags are pretty much the same in regular browsers. However, there is a key difference.

    I stands for italic, and looks like this. I is simply used to emphasize text to the naked eye, something a blind person might not understand if they were having the text read to them.

    Em, on the other hand, stands for emphasize, and looks like this. Em is used in order to indicate how text should be understood. The em tag will manifest itself in different ways, such as sounding different in an interpreter software for blind people.

  • b and Strong Tags

    Much like em and i, b and strong may look the same but have a key difference.

    B stands for bold and looks like this. It is only used to make text bold to the naked eye.

    Strong looks like this and indicates how text should be understood. Like em, strong text might be read differently in interpreter software.
